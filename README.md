Picxima
===
A website specifically designed for photo editing.

Unlike other photo editing websites, Picxima offers you a minimalist and highly secure editor.

Written by the "three old men of the web" (HTML, CSS, JS). This makes it accessible from anywhere, regardless of the device you're using. 

Featured
--------
- Convert the original image format to various other formats.
- Compress images to reduce file size.
- And more features in the future!
